# 기말 프로젝트 정리사항
1. 프로젝트 목표

손글씨가 촬영된 동영상을 YOLO로 감지한 후, 숫자가 인식되면 해당 이미지를 PGM 파일로 저장하고, 이를 MNIST 인식 예제의 입력으로 사용하여 정확도를 향상시키는 것이 목표입니다.

2. 입력 데이터
손글씨 동영상
영상에는 약 10~15개의 숫자가 무작위 순서로 등장
촬영 방식에는 제한이 없습니다.

예시)
6 (2초) → 8 (3초) → 8 (1.5초) → 6 (2초)

3. YOLO 요구사항
필수 기능
손글씨 숫자를 YOLO로 감지해야 합니다.
숫자가 감지되면 해당 영역을 28×28 PGM 파일로 저장해야 합니다.
사용 언어, 학습 데이터셋, 프레임워크에는 제한이 없습니다.

PGM 저장 규칙
일반적인 YOLO는 동일한 숫자가 화면에 존재하는 동안 여러 프레임에서 반복 검출됩니다.
그러나 본 프로젝트에서는 다음과 같이 처리해야 합니다.

예시)
6 (2초) → 8 (3초) → 8 (1.5초) → 6 (2초)

저장되어야 하는 PGM 파일 수:
6, 8, 8, 6
총 4개
즉, 동일 숫자가 연속해서 검출되더라도 해당 숫자가 화면에 등장한 구간당 1개의 PGM 파일만 저장해야 합니다.

MNIST 평가 데이터
YOLO가 생성한 PGM 파일 중 처음부터 손글씨 등장 개수(10~15개)만 평가합니다.
예를 들어 손글씨는 12개였지만 PGM 파일이 20개 생성되었다면, 처음 12개만 평가합니다.

4. MNIST 요구사항
YOLO가 생성한 PGM 파일을 입력으로 사용합니다.
기존 MNIST 코드의 함수 호출 순서(구조)는 수정할 수 없습니다.
단, 함수 내부 최적화는 가능합니다.
기존 숫자 1, 3, 5의 인식 성능은 유지해야 합니다.
새롭게 추가되는 숫자 6, 8의 인식 성능을 향상시켜야 합니다.
학습 데이터는 외부 데이터셋 또는 제공된 손글씨 데이터를 자유롭게 사용할 수 있습니다.

목표는 최대한 높은 정확도 및 달성하는 것입니다.

5. 시연 방법
① YOLO 단계
입력: 손글씨를 칠판에 붙여놓고 촬영한 동영상
동작: 영상을 입력받아 숫자를 감지합니다. 숫자가 새롭게 등장했다고 판단되면 PGM 파일을 저장합니다.
② MNIST 단계
입력: YOLO가 생성한 PGM 파일
동작: 저장된 PGM 파일을 입력으로 받아 숫자를 인식합니다. 수행시간은 아래 8번의 기준에 따라 측정합니다.

9. 수행시간 측정 규칙

다음 측정 위치를 반드시 유지해 주시기 바랍니다.

모든 변수 초기화 완료 후 측정 시작
cudaDeviceSynchronize() 직후 측정 종료

예시코드)
checkCudaErrors(cudaMemcpy(srcData, imgData_h,
                           IMAGE_H*IMAGE_W*sizeof(value_type),
                           cudaMemcpyHostToDevice));

n = c = 1;
h = IMAGE_H;
w = IMAGE_W;

// 시작
auto infer_start = std::chrono::high_resolution_clock::now();

convoluteForward(conv1, n, c, h, w, srcData, &dstData);
poolForward(n, c, h, w, dstData, &srcData);

convoluteForward(conv2, n, c, h, w, srcData, &dstData);
poolForward(n, c, h, w, dstData, &srcData);

fullyConnectedForward(ip1, n, c, h, w, srcData, &dstData);
activationForward(n, c, h, w, dstData, &srcData);
lrnForward(n, c, h, w, srcData, &dstData);

fullyConnectedForward(ip2, n, c, h, w, dstData, &srcData);
softmaxForward(n, c, h, w, srcData, &dstData);

checkCudaErrors(cudaDeviceSynchronize());

// 종료
auto infer_end = std::chrono::high_resolution_clock::now();

double infer_ms =
    std::chrono::duration<double, std::milli>(
        infer_end - infer_start
    ).count();

std::cout << "Inference time: "
          << infer_ms
          << " ms" << std::endl;

다른 시간 측정 함수를 사용해도 무방하지만, 반드시 동일한 측정 구간을 유지해야 합니다.
